# Iskonnect

A scholarship management mobile application for City of Educ scholars and applicants.

## Features

### For Scholars
- 📢 View announcements
- 📚 Track subjects and grades by semester
- 📋 Submit requirements (COR, documents)
- ✅ Track attendance
- 📊 View GWA and academic progress

### For Applicants
- 📝 Apply for scholarship
- 📄 Upload required documents
- 🔍 Track application status
- 📢 View important announcements
